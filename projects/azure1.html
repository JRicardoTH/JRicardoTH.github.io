ML Deployment with Azure's Notebooks, AutoML & Design

This project explores the deployment of Decision Tree and Random Forest classifiers using Azure's Notebooks, AutoML, and additional design tools for model comparison.

The problem involves predicting bank subscription based on customer behavior. The dataset was cleaned, processed, and fed into Azure pipelines for automated evaluation.

The deployment workflow includes:

  • Data preprocessing in Python notebooks
  • Model comparison using Azure AutoML
  • Deploying the best model as an API endpoint
  • Testing the deployed inference endpoint
